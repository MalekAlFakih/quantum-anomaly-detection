.. _installation:

Installation
============

To install Quantum Anomaly Detection, clone the repository:

.. code-block:: bash

   git clone https://github.com/your-username/quantum-anomaly-detection.git
   cd quantum-anomaly-detection
   pip install -e .

Requirements
------------

- numpy
- qiskit
- matplotlib
- jupyter

.. code-block:: bash

   pip install -r requirements.txt
